# SentinelCore
一个All in one的边缘智能网关，该网关能够实现：
1. 采集多种下挂设备（串口、IIC、TCP）数据。
2. 在本地进行数据处理与存储。
3. 通过MQTT协议将数据安全上报至云平台（阿里云服务器）。
4. 支持远程控制下挂设备以及网关自身的配置更新和OTA升级。
5. 可以通过Web服务和应用来优雅的展示数据和控制。

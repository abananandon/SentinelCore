# SentinelCore
一个All in one的家庭监控中枢，能够对家庭环境进行监测，能够控制家庭里的各种只能设备，并且其状态展示在Web客户端上。

# What can do？
- [ ] 获取系统运行时的信息，包括CPU利用率与温度，内存使用率等
- [ ] 检测环境信息，包括温度、湿度、光照强度等
- [ ] 连接和控制蓝牙智能设备，包括智能插座、灯光、空调等等
- [ ] 将上述信息以Web客户端的形式展现出来，并远程控制
